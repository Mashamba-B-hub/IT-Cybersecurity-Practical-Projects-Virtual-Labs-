## Overview

This folder contains screenshots captured during the Wireshark network analysis lab. Each image provides visual evidence of identified packets and protocol behaviour.

## Screenshot Index

* **arp-request.png**
  Displays an ARP request packet showing source and destination MAC/IP addresses.

* **arp-reply.png**
  Shows the ARP reply confirming MAC address resolution.

* **ping-request.png**
  Captures an ICMP Echo Request (Type 8).

* **ping-reply.png**
  Captures an ICMP Echo Reply (Type 0).

* **dns-query.png**
  Displays a DNS query packet requesting domain resolution.

* **dns-response.png**
  Shows the DNS response with the resolved IP address (A record).

* **tcp-handshake.png**
  Illustrates the TCP three-way handshake (SYN, SYN-ACK, ACK).

* **https-traffic.png**
  Displays HTTPS traffic over port 443, showing encrypted data transmission.

## Notes

* Screenshots were captured directly from Wireshark during live traffic analysis
* Each image highlights key protocol details and packet structures


## Purpose

These screenshots serve as proof of packet capture, filtering, and protocol analysis performed in the lab.
